# Placement Portal 🎓

This is a Flask-based web application to manage placement-related functionalities for students and coordinators.

## 🔧 Features
- Student and Teacher login
- Profile management
- Attendance and grades tracking
- Admin dashboard
- Modern UI (black & gold theme)

## 🚀 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- MySQL
- Bootstrap / Custom CSS
